# FunWithRSAAlgorithm

### Overview
A simple file to demonstrate the RSA Algorithm in Python

### To run
* Clone the repo
* cd into it
* run the file from the command line with ```$ python RSAAlgorithm.py```
* respond to user_input request
* see demo of encryption and decryption

### Pre-requisites
* Python 3.8
